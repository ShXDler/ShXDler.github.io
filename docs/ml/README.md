Chapter 2

P39 及之后的样本均值和样本方差符号应当为$\bar X$和$S^2$，文中使用的符号用于总体均值和总体方差

P39 使用t分布前应当说明$\epsilon$近似服从正态分布

P41 McNemar检验中$|e_{01}-e_{10}|$似乎并不服从正态分布，该统计量之所以服从卡方1的分布是因为皮尔逊卡方公式用极大似然法估计了两个值，自由度为4-2-1=1

P45 方差、偏差符号中的$x$应当替换为$f(x)$
